LinkedArray

This is a priority queue implementation that uses a linked list of arrays as the base structure. It starts with an array of size one, then once that is full, creates an array of size two and adds it to the linked list (this array will hold the previous ones children). Once that is full, it will create and array of size 4 (the array sizes grow at a rate of 2^n) which will hold the previous array's children, and so on. This allows for better memory management, as each the largest array is (worst case) only half the size of the total array size from a standard implementation.
